title: "str reserved keyword in Python"
categories: [Python]
---

<div markdown="1" class="ans">
`str` is a special function representing the string class.
</div>

<div markdown="1" class="ans">
```python
s = str(obj)
```
</div>

`str` can be used to convert `obj` to string.
