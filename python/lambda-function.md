---
title: Lambda functions in Python
---

<div markdown="1" class="ans">
```python
pow2 = lambda x: x*x
```
</div>

### Calling
```python
x = pow2(3) # x will receive 9
```

### Related names from other languages

- **functor** from C++
- **delegate** from C#
- **arrow function** from JavaScript
- also **lambda expression** from Java and C#
